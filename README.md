# AI-Driven-Swarm-IoT-Bird-Deterrent-System-with-Predictive-Adaptive-Crop-Protection

## Project Overview
AI-powered IoT system for smart crop protection. Uses computer vision on Raspberry Pi to detect birds and triggers ESP32-based deterrent devices like buzzers or motors. Designed with adaptive and predictive monitoring for efficient bird control in agricultural fields.

## Components Used
- Raspberry Pi
- USB Webcam
- ESP32
- Buzzer
- Motor Driver

## Features
- Real-time bird detection
- Live camera monitoring
- Buzzer alert system
- Motor activation to scare birds

## Technologies
- Python
- OpenCV
- Flask
- ESP32 (Arduino)

## Future Improvements
- Drone-based surveillance
- AI model using YOLO
- Night vision monitoring
